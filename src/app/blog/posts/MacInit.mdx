# Mac 基础设置

<InlineCode>系统设置 > 搜索："触发角快捷键" > 左下角"锁定屏幕"</InlineCode>

<InlineCode>系统设置 > 键盘 > 关闭"听写-快捷键"</InlineCode>

# NVM✨

[下载 Zip 压缩包](https://github.com/nvm-sh/nvm) 或 使用 homebrew 安装
<CodeBlock className="my-24"
    compact
    codeInstances={[
        {
            code: 'brew install nvm',
            language: 'bash'
        },
    ]}
    copyButton
/>
通过 <InlineCode>shift+command+.</InlineCode> 查看隐藏文件，将解压后的结果改成.nvm，然后再放入用户根目录

然后对zsh进行配置：
<CodeBlock className="my-24"
    compact
    codeInstances={[
        {
            code: '# NVM\nexport NVM_DIR="$HOME/.nvm"\n[ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"  # This loads nvm\n[ -s "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"',
            language: 'bash'
        },
    ]}
    copyButton
/>
最后 <InlineCode>source ~/.zshrc</InlineCode>

nvm相关命令
<CodeBlock className="my-24"
    compact
    codeInstances={[
